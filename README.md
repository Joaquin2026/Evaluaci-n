# Tienda de videojuegos

Descripción
- Proyecto de interfaz simple para una tienda de videojuegos. Enfocado en semántica HTML5 y uso de Flexbox para el layout.

Estructura de carpetas
- / (proyecto)
  - index.html
  - README.md
  - /css
    - estilos.css
  - logo.png

Prompts usados para crear el HTML y CSS
 EL index.html
1. Genera un documento HTML5 semántico para un sitio web de una Tienda de videojuegos. Debe incluir: un header con logo, un nav con 3 links, un main con una sección de bienvenida y otra con información de contacto, y un footer. Usa solo etiquetas semánticas de HTML5


El estilos.css
1. Crea estilos profesionales para este sitio. Usa Flexbox para centrar el contenido y para que el menú de navegación sea horizontal. Asegúrate de que el diseño sea ordenado, con una paleta de colores coherente y que el formulario tenga un estilo moderno. Usa CSS externo.

Objetivo técnico
- Mantener semántica HTML5 (header, nav, main, section, footer, address).
- Usar Flexbox para el layout del header y elementos de navegación, con adaptaciones responsivas mediante media queries.

Instrucciones rápidas
- Abrir index.html en un navegador o servir desde un servidor local (por ejemplo: Live Server en VS Code).
- Editar css/estilos.css para ajustar colores, tipografía o espaciamientos.
